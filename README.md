# Фильмдермен жұмыс жасау бағдарламасы (8–12 апта тапсырмалары)

Бұл жоба Python тілінде жазылған консольдік бағдарлама.  
Фильмдер тізімін басқару, файлға сақтау, қателерді өңдеу, рекурсия,  
сонымен қатар OOP, NumPy және Matplotlib тақырыптарын қамтиды.

## Бағдарлама мүмкіндіктері

- Фильмдер тізімін көру  
- Жаңа фильм қосу  
- Фильмді жою  
- Рейтингін көру  
- Мәліметтерді файлға сақтау (data.txt)
- Файлдан оқу  
- Қате енгізуді өңдеу (try-except)
- Рекурсия мысалы (ask_repeat функциясы)

## Пайдаланылған модульдер мен функциялар

### utilis.py
- add_film(films, name, rating) — жаңа фильм қосу  
- delete_film(films, name) — фильмді жою  
- find_film(films, name) — фильмді табу  

### sss.py (негізгі бағдарлама)
- show_films() — фильмдердің тізімін көрсету  
- save_to_file() — фильмдерді файлға жазу  
- load_from_file() — файлдан оқу  
- menu() — негізгі мәзір  
- ask_repeat() — рекурсиямен қайта сұрау
